openPOWERLINK - MN - QT - GUI

Version: 1.1.0

Source: https://github.com/Ramakrishnan45/openPOWERLINK_v2.0_QT.git

Package dependencies to compile

Compiler

Windows: Microsoft visual C++ 2010
Linux: G++ v4.6.3

Qt

Windows: Download and install Qt 5.2.1 for Windows
Linux: Download and install Qt 5.2.1 for Linux

Build source

  1. Build the openPOWERLINK STACK libraries.
  2. Build the plk_qt_api present in demo_qt_advanced directory
    1. open plk_qt_api.pro using qt-creator
    2. Build the project
    3. After the successful build PlkQtApi.dll will be generated in release/debug directory
  3. Build the plk_qt_gui present in the demo_qt_advanced directory
    1. open plk_qt_gui.pro using qt-creator
    2. Build the project(ctrl+B).
    3. After the successful build plk_qt_gui will be generated in release/debug directory
    4. From qt-creator run the application(F5).
